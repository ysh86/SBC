
        =================
        Universal Monitor
        =================

Universal Monitor is a simple monitor program for microcomputer systems.


Supported processor
-------------------

    o Motorola MC6800 (including MC6802 and MC6801/3)
    o Motorola MC6809
    o Intel 8080
    o Atmel AVR
    o Zilog Z8
    o Zilog Z80
    o MOS Technology 6502
    o Toshiba TLCS-90
    o National Semiconductor SC/MP
    o Panafacom MN1610
    o Texas Instruments TMS9900
    o Intel 8086
    o Intel 8051
    o Fairchild F8
    o Zilog Z8000
    o Motorola MC68000
    o Hitachi SuperH
    o Hitachi H8/300H
    o RCA COSMAC
    o National Semiconductor INS8070 (SC/MP III)
    o NEC uCOM-87
    o Signetics 2650
    o Intersil IM6100
    o GI CP-1600 (Contribution by Kasumi Yoshino)
    o Hitachi H8/500
    o National Semiconductor NS32000


Requirements
------------

    o Build

        - The Macroassembler AS
        - make

    o Host (terminal)

        - Serial port
        - Terminal software

    o Target

        - 2kB ~ 4kB ROM (depends on target processor)
        - 64B ~ 256B RAM (depends on target processor)
        - Serial port

        or

        - EMILY Board
